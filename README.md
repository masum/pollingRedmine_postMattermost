# 実行

```
$ git clone {}
$ npm install
$ export REDMINE_URL={Redmine URL}
$ export REDMINE_PROJECT={Redmine Project Name}
$ export REDMINE_KEY={Redmine Key}
$ export REDMINE_PROJECT_ID={Redmine Project ID}
$ export REDMINE_ICON={Redmine Icon URL}
$ npm start
```
